# Beacon

Beacon is an independent, portable Windows launcher being rebuilt with **WinUI 3**.

The project focuses on a fast keyboard-first search experience for applications, files, folders, settings, and actions. The first distribution target is a self-contained portable ZIP that can run without an installer or a preinstalled Windows App SDK runtime.
